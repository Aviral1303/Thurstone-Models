Method. A race is a resolved YES/NO market with ≥2 traders holding positions; performance is realized return per mana ((payout + sale proceeds) ÷ mana bought). Each race decomposes into pairwise duels (tie when |Δlog r| ≤ 0.01); abilities are per-vote MLE under the half-tie scheme. 95% intervals: cluster bootstrap over 1,081 underlying clusters, B = 1,000, zero failed replicates.

Robustness. Kendall τ vs Bradley–Terry 0.995; across lattice units 0.995–0.997; across tie bands ε = 0.005/0.02, 0.981–0.985.

Caveats. θ scores duel wins, not profit — a rare-big-win style can bank mana while losing most duels. Our bots run in a fixed daily execution order, so realized returns include queue position. Snapshot 2026-07-14 16:03; 6,096,312 bets back to platform genesis (2026-02-18).
